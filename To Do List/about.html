The To-Do List App is a simple yet functional web application designed to help users manage their tasks efficiently. Built using HTML, Javascript, and styled with Tailwind CSS, the app allows users to add tasks, mark them as completed, and delete them as needed.
Key Features :
1. Task Creation: Users can easily add new tasks using an input field and a button. Each task is displayed as a list item.
2. Task Deletion: A "Delete" button is available for each task, enabling users to remove tasks once completed or no longer needed.
3. Responsive Design*: Tailwind CSS ensures that the app is visually appealing and responsive across different screen sizes.
The app is a great demonstration of core web development concepts like DOM manipulation, event handling, and responsive design, making it a useful project for beginners in web development.
